Git cvičení

  1. Vytvoř nový veřejný repozitář na GitHubu s názvem git-cviceni.

  2. Repozitář k sobě naklonuj.

  3. Do nové složky přidej soubor index.html s nadpisem <h1>Git cvičení</h1>.

  4. Kód commitni a pushni.

    1. git add .
    2. git commit -m "Základní html"
    3. git push

  5. Zkontroluj si, že index.html se nahrál na GitHub.

  6. Přidej soubor se styly. V html ho napoj.

  7. Na body veškerému textu nastav bezpatkové písmo, zarovnání na střed a maximální šířku na 600px.

    body {
    	font-family: sans-serif;
    	text-align: center;
    	max-width: 600px;
    	margin: 0 auto;
    }

  8. Změny commitni se zprávou "Základní styly".

  9. Nadpis obal do tagu <header>, který podbarvi barvou #166088. Text udělej bílý.

  10. Commitni.

  11. Přidej do html pod hlavičku sekci <main> s pěti odstavci textu.

    main>p*5>lorem

  12. Velikost textu v odstavcích dej na 20px a odsaď ho o zhruba 60px od horního nadpisu a konce stránky.

    main {
    	padding: 3em 0;
    	font-size: 20px;
    }

  13. Commitni.

  14. Přidej patičku <footer>Patička 2021</footer>.

  15. Nastav jí pozadí na #dbe9ee.

  16. Commitni.

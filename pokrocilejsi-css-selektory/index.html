Pokročilejší CSS selektory

  • *: všechny prvky
  • + a ~: sousední prvky
  • :nth-child (:first-child, :last-child, :nth-of-type): prvky na základě pořadí
  • >: přímí potomci
  • :has: prvky, které obsahují jiné prvky
  • :empty: prázdné prvky
  • :is: skupina selektorů
